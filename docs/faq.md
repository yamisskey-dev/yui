# よくある質問・トラブルシュート

## Q. 天気noteが投稿されない／遅い
- サーバが起動しているか確認
- 天気APIの障害やネットワーク不調の可能性あり
- ログにエラーが出ていないか確認

## Q. 設定変更が反映されない
- `npm run build` → `docker compose build` → `docker compose up -d` の手順を必ず実施
- 開発時は built/ のマウントでホットリロードも活用

## Q. バージョンタグを間違えて付けた
- `git tag -d vX.Y.Z` でローカル削除
- `git push --delete origin vX.Y.Z` でリモート削除

## Q. その他のトラブル
- ログ出力・エラーメッセージを確認
- 不明な場合はissueや管理者に相談
